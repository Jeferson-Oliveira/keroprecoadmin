instalar scene builder

instalar git
	após isso selecione "clone repository" e selecione o keroprecoadmin
	será baixada uma cópia do projeto e colocado no repositório (C:Users\[seu usuario]\Documents\GitHub\

instalar postgreSql
	ao instalar o PostgreSql quando pedir pra definir uma senha coloque "postgres" (sem aspas)
	ao terminar procure por PGAdmin na aba de aplicativos e inicie
	selecione file -> preferences -> binary
	coloque nos campos o endereco de instalação do postgres ex: C:\Program Files\PostgreSQL\9.6\bin e depois salve
	expanda a aba servers -> postgres (vai pedir a senha, vc coloca "postgres")
	clique com o botão direito em cima do nome "postgres" e selecione a opção "restore"
	no campo do diretório coloque o caminho do arquivo "keroprecodb" que o git baixou lá na pasta documents ex: C:Users\[seu usuario]\Documents\GitHub\keroprecoadmin\keroprecodb 
	clique em restore e aguarde completar com sucesso

importar projeto no netbeans
	abra o netbens e selecione a opção "abrir projeto"
	vá até o diretório que se encontra o projeto baixado no git (Documents) e selecione ele
	após importar rode o projeto (login e senha "admin")
